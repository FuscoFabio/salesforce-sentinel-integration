# Caso d'Esempio: Azure Function

Esempio completo di implementazione Azure Function per integrazione Salesforce-Azure Sentinel.

## Scenario

Un'organizzazione ha bisogno di personalizzazioni avanzate:
- Trasformazione dati custom
- Integrazione con altri sistemi
- Filtraggio eventi specifici
- Enrichment dati esterni

## Requisiti

- Licenza Salesforce Enterprise
- Azure Subscription
- Azure Function App
- Competenze di sviluppo
- Tempo setup: 1-2 giorni

## Implementazione Completa

Vedi la guida dettagliata: [Implementazione Azure Function](../implementation/azure-function.md)

## Caratteristiche Personalizzate

- Polling configurabile (5-15 minuti)
- Trasformazione dati custom
- Integrazione con API esterne
- Filtraggio eventi avanzato

## Risultati Attesi

- Eventi disponibili in Azure Sentinel dopo 24-48 ore
- Dati trasformati secondo logica custom
- Integrazione con sistemi esterni
- Monitoraggio e alerting personalizzati
