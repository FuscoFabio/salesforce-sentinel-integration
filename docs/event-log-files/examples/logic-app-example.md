# Caso d'Esempio: Logic App

Esempio completo di implementazione Logic App per integrazione Salesforce-Azure Sentinel.

## Scenario

Un'organizzazione preferisce un approccio low-code visuale:
- Workflow visuale
- Connettori integrati
- Template predefiniti
- Configurazione grafica

## Requisiti

- Licenza Salesforce Enterprise
- Azure Subscription
- Azure Logic App
- Tempo setup: 30-60 minuti

## Implementazione Completa

Vedi la guida dettagliata: [Implementazione Logic App](../implementation/logic-app.md)

## Caratteristiche

- Workflow visuale
- Connettori nativi
- Template ARM
- ISE per sicurezza

## Risultati Attesi

- Eventi disponibili in Azure Sentinel dopo 24-48 ore
- Workflow configurabile visualmente
- Integrazione con altri connettori Azure
- Monitoraggio e alerting
