# Caso d'Esempio: Azure Event Hub

Esempio completo di implementazione Azure Event Hub per integrazione Platform Events con Azure Sentinel.

## Scenario

Un'organizzazione enterprise ha bisogno di:
- Monitoraggio near real-time
- Volume eventi elevato
- Alta affidabilità
- Persistenza eventi

## Requisiti

- Licenza Salesforce Enterprise
- Azure Subscription
- Azure Event Hub
- Azure Function App
- Competenze sviluppo Apex e Azure
- Tempo setup: 3-4 giorni

## Implementazione Completa

Vedi la guida dettagliata: [Implementazione Event Hub](../implementation/event-hub.md)

## Caratteristiche

- Latenza < 1 minuto
- Scalabilità automatica
- Persistenza eventi
- Alta affidabilità

## Risultati Attesi

- Eventi disponibili in Azure Sentinel in < 1 minuto
- Gestione automatica picchi
- Persistenza eventi garantita
- Scalabilità enterprise
